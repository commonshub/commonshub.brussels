# Elinor

Elinor is an AI community manager running on the [Commons Hub Brussels Discord](https://discord.commonshub.brussels) via [OpenClaw](https://openclaw.com).

## What She Does

- Answers community questions about events, rooms, and the space
- Provides event information from the data pipeline
- Helps with room booking inquiries
- Welcomes new members and facilitates introductions

## Technical Details

- Runs as an OpenClaw agent connected to Discord
- Has access to the Commons Hub data (events, rooms, settings)
- Named after [Elinor Ostrom](https://en.wikipedia.org/wiki/Elinor_Ostrom), Nobel laureate and inspiration for the Ostrom Room
